Detailed Description

The Election class is an abstract class that provides the necessary functionality to represent an election. While non-abstract child classes are required to implement their own Ballot tallying algorithms and results, the Election class provides a template of necessary methods and properties that all types of elections will utilize.

The constructor for the Election class sets the type of the election, the number of seats in the election, the vector that stores the Candidates who are running in the election, and the vector that stores the Ballots cast in the election. It also defaults the name of the audit file path to the empty string.

Detailed Description

The Ballot class provides the necessary functionality to represent a voter's ballot in an election. It is able to represent ballots in both plurality and single-tranferrable vote elections. A ballot had an Id, a list of the Candidates in the order they were ranked on the ballot, and a variable indicating the current highest ranked Candidate choice remaining on the ballot.
